# My setup

From a brand new Ubuntu installation, run:
```
cd
git clone https://github.com/mynameuhh/dotfiles.git
cd dotfiles
./newComputerSetup.sh
```
Then `reboot` and open with i3.

## Installs
- i3 + my i3 config
- Git + my git aliases
- Python3 + pip3
- Redshift
- Discord
- Spotify
- VS code
- Firefox
- neovim + my config + git default editor + alias to `vim`
